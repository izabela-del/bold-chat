# Bold Chat

Proof-of-concept for the Bold care chat experience.

## Contents

- `bold-care-chat-poc.html` — the interactive prototype (open in a browser)
- `_c*.js`, `_check.js` — supporting scripts / iterations
- `*.png` — UI mockups and reference screens (landing, onboarding, coverage, health, etc.)

## Running

Open `bold-care-chat-poc.html` directly in a browser, or serve the folder locally:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000/bold-care-chat-poc.html
```
